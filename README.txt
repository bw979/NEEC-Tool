# NEEC-Tool
This shiny app is to assist with NEEC candidate selection and provides a surface for NEEC rates due to CSD for Te < 100keV. Candidate NEEC rates can also be assessed as an upper limit for temperatures above this

Run and deploy on a server via app.R. If there are problems then you normally need to install the missing packages.

Instructions for running calcultions are displayed on the app itself.

Do not reproduce this app or results without consulting ben.wallis@york.ac.uk
